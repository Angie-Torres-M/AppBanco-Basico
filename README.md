
# Proyecto Banco en C#

Sistema bancario desarrollado en C# como práctica de Programación Orientada a Objetos (POO) y estructura de proyectos en .NET. Incluye creación de cuentas, depósitos, retiros, transferencias y validaciones básicas de saldo.

---

## Funcionalidades

- Crear cuentas bancarias
- Consultar saldo
- Realizar depósitos
- Realizar retiros con validación
- Transferencias entre cuentas
- Historial básico de operaciones (opcional)
- Uso de clases, objetos, encapsulamiento y métodos

---

## Instalación y ejecución

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Angie-Torres-M/AppBanco-Basico.git
````

2. Abre el proyecto:

   * Visual Studio
     o
   * VS Code con extensión de C#

3. Ejecuta:

   ```bash
   dotnet run
   ```
